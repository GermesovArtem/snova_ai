# Памятка по ошибке 422 (KIE.AI)

Если KIE API возвращает `code: 422` ("The model name you specified is not supported..."), необходимо проверить следующие пункты:

1. **Название модели**: Kie очень чувствителен к регистру и символам. `Nano-Banana-2` или `nanobanana2` вызовут ошибку 422. Единственно верный формат (через дефис в нижнем регистре) — `nano-banana-2`.
2. **Структура input**: Ошибка 422 также вылетает, если в поле `input` переданы неподдерживаемые параметры (например, если модель не поддерживает `resolution: 4k`, а мы его передали).
3. **Тип данных**: Убедиться, что `model` передается строго как строка (`string`), а не как вложенный объект.
